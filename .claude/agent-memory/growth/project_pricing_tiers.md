Three tiers established in the growth playbook (2026-03-22):

1. **Visibility Assessment** (one-time): Mid-market $7,500 / Enterprise $15,000 / Multi-brand $25,000
2. **Visibility Intelligence** (annual): Mid-market $30K/yr / Enterprise $60K/yr / Multi-brand $100K/yr
3. **Strategic Visibility Program** (annual, anchor): Enterprise only $150K-$250K/yr

Key pricing decisions:
- Value-based, not cost-plus
- Annual billing default (monthly at 20% premium for mid-market only)
- 5-8% annual price escalation built into contracts
- "3 visibility gaps or it's free" guarantee on Assessment tier
- Strategic tier serves primarily as price anchor, but is deliverable

**Why:** Pricing follows Hormozi principle that price is the most powerful profit lever. Tiers map to buyer maturity (understand position -> monitor ongoing -> own the narrative).

**How to apply:** Use these tiers and prices as baseline for all proposal language, sales deck content, and landing page pricing sections. Adjust only if founder explicitly changes pricing strategy.
