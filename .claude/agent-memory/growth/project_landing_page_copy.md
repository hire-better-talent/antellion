Landing page copy deck written at `/docs/designs/landing-page-copy.md` on 2026-04-08. Covers the full page architecture for Antellion's lead capture page targeting VP-level TA and employer brand leaders.

Key decisions:
- 9-section flow: Hero -> Problem -> What You Get -> How It Works -> Example Finding -> Credibility -> Lead Capture Form -> FAQ/Objections -> Footer
- Headline: "What does AI tell candidates about your company?" (question format, not a claim)
- Primary CTA: "Get Your Visibility Snapshot" / "Run My Snapshot" (on the form button)
- Lead capture fields: name, work email, company name, company website (required); title, biggest talent competitor, primary role hired for (optional)
- 48-hour delivery expectation set throughout
- No pricing on the page, no self-service option, no dashboard screenshots
- Example finding based on Meridian Technologies seed data (anonymized)
- Warm traffic (LinkedIn DM recipients) and cold traffic (organic) both considered in the design
- Language rules aligned with snapshot outreach guide and category positioning docs
- No fake testimonials, logos, or statistics -- credibility comes from methodology specificity

**Why:** The landing page is the first public-facing asset for Antellion. It must work for both cold traffic and warm traffic (DM recipients clicking through to verify legitimacy). The page is NOT self-serve -- it captures lead info for manual snapshot production.

**How to apply:** Reference this document when building the actual page. All copy is production-ready. Visual direction notes guide the frontend implementation. The form field spec and confirmation copy are ready to implement as-is.
